# Gen AI Academy APAC Edition - Hackathon Submissions

Submissions for the Gen AI Academy APAC Edition (Hack2Skill) - Cohort 1 Academy Tracks.

## Track 1: Build and deploy AI agents using Gemini, ADK, and Cloud Run

**Project:** Smart Text Assistant

A single AI agent built with Google ADK and Gemini 2.5 Flash, deployed on Cloud Run. The agent performs text summarization and priority classification through an HTTP endpoint.

- **Agent:** `track1_smart_assistant/`
- **Tech:** Google ADK, Gemini 2.5 Flash, Vertex AI, Cloud Run
- **Deployment:** `adk deploy cloud_run`

## Track 2: Connect AI agents to real-world data using MCP

**Project:** News Intelligence Agent

An AI agent built with Google ADK that uses MCP (Model Context Protocol) to connect to external data sources, retrieve structured information, and generate intelligent responses.

- **Agent:** `track2_news_agent/`
- **Tech:** Google ADK, Gemini 2.5 Flash, MCP Toolbox for Databases, BigQuery, Cloud Run
- **Deployment:** `adk deploy cloud_run`
